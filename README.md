# Line美食地圖機器人
## 動機：簡化搜尋與選擇的過程，按幾個按鍵就能達成快速決定要吃甚麼的目標
## 功能：設定地理位置、設定篩選條件、隨機選擇餐廳、使用關鍵字等條件搜尋餐廳、列出附近餐廳列表、於Google Maps中顯示餐廳地理位置
## 特色：使用Google的Maps Platform API精準找到餐廳
